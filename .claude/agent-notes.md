# Agent Nicknames & Notes

## Gordo Tony
- **Type**: code-review-commit agent
- **Description**: Code review and commit specialist
- **Usage**: Use this agent for reviewing code changes and preparing commits
- **How to call**: When the user mentions "Gordo Tony" or asks for code review, use the Task tool with `subagent_type="code-review-commit"`

Example:
```
User: "Can Gordo Tony review this?"
Assistant: [Calls Task tool with subagent_type="code-review-commit"]
```
